# Changelog

All notable changes to the Anti-Gravity Solar System project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Navigation system with FreeFlight, Orbit, and Focus modes
- Procedural planet generation with noise-based terrain
- Advanced visualization for gravity fields and forces
- UI panels for control, object info, camera control, presets, debug, and status bar
- Collision detection system
- Energy monitoring system
- Orbit trails rendering
- Multiple preset configurations (Solar System, Binary Star, Chaos, Anti-gravity)

### Changed
- Improved camera controls with smooth transitions
- Enhanced physics engine performance
- Updated documentation structure

### Fixed
- Various stability improvements
- Memory leak fixes in rendering system

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- N-body gravitational simulation
- Anti-gravity mode for repulsive forces
- Real-time physics simulation
- Interactive UI controls
- Debug visualization tools
- Time scale manipulation
- Body selection and inspection
- Preset management system
- Procedural planet texturing
- Navigation system for space exploration
- FPS counter and performance metrics
- Cross-platform support (Windows, macOS, Linux)

### Technical Details
- Built with C++17
- OpenGL 2.1 rendering
- ImGui for user interface
- GLFW for window management
- GLM for mathematics
- JSON configuration files
- CMake build system

## [0.1.0] - Development Phase

### Added
- Core physics engine
- Basic rendering pipeline
- Initial UI framework
- Solar system data loader
- Basic celestial body mechanics
